# Подсказка по гиту

```sh
cd "Путь файла" - выбирает папку в которой будут происходить изменения

git log — вызывать список действий и сохранений,

git init — создать репозиторий в папке на локальной машине,

git add — отслеживать добавленные файлы,

git commit + комментарий — сохранить текущее состояние,

git diff — показать разницу между версиями,

git chechout — переключиться между разными версиями.

git checkout master - возврат на саму последню версию.

git branch - просмотр веток гита

git merge "имя ветки" - обьединяет вет в которой вы находитесь с веткой которую вы вводите вместо "имя ветки" без кавычек 

git log --oneline - показывает комиты в укороченом варианте

git log --graph - показывает комитты и их разветвления

git push - перекидывает изменинея с компа в гитхаб

git pull - перекидывает изменения с гита на комп\

git clone "Ссылка" - клонирует репозиторий  

```
